# What is Ampersand???
Ampersand is a graphics API written in Rust for X11 Linux. It prioritises memory safety and efficiency.
Right now, it is a simple GTK3 wrapper. However, in future (before MAJOR release 1), it will, rather than wrap GTK, be its own library using Cairo.
For more info, see [docs/roadmap.md](./roadmap.md)

# What to expect
I want this personal project to be the best it can be. However, I am by no means a great coder. Ideally, this project will be expanded into something not just operational, but genuinely useful. Don't expect the greatest graphics API to ever see the internet, instead expect a single hobbyist developer attempting to make something probably far out of scope of his knowledge. Contributions, advice and criticism is greatly appreciated.
